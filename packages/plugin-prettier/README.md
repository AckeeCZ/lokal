# @lokse/plugin-prettier

Plugin used to format output translations file using Prettier.

## Installation

```sh
$ yarn add -D @lokse/plugin-prettier
```

## Usage

Add it into plugins section of lokse config

```json
{
    "plugins": ["@lokse/plugin-prettier"]
}
```
